# Darex ile Gücünüze Güç Katın:
- Performans: Automated Cache Engine: Darex, klasik botların aksine Roblox API'sini her işlemde yormaz. Gelişmiş Bellek İçi (RAM) Önbellekleme mimarisi sayesinde: Tepki Hızı: 300ms'den <5ms seviyesine düşürüldü.
- Çoklu Grup ve Gelişmiş Mimarisi: Tek bir Discord sunucusuna hem ana grubunuzu hem de ek rütbe gruplarınızı bağlayarak hepsini tek bir dinamik panelden, karmaşa olmadan yönetin.
- Gelişmiş Rütbe Yönetimi (/rütbe): Roblox web sitesine girmeden, Discord üzerinden kullanıcı adı ve sebep belirterek saniyeler içinde rütbe ataması yapın. Sayfa değiştirmeli rütbe havuzu ve Premium sunucular için Rütbe ID ile manuel atama desteği!
- Kitlesel Yönetim (/toplu-rütbe): Belirli bir Discord rolüne sahip olan tüm üyeleri tek komutla topluca hedef Roblox rütbesine atayın. API sınırlarına takılmayan akıllı kuyruk motoruyla toplu operasyonları sorunsuz yürütün.
- Grup Koruma Kalkanı (/grup-yedek): Olası yetkili suistimallerine (Abuse) karşı felaket kurtarma senaryomuz aktif! Grubun rütbe hiyerarşisini tek tıkla MongoDB'ye yedekleyin; kriz anında tek komutla tüm grubu eski haline geri yükleyin!
- Kurumsal Denetim (/yetkili-denetle): Yetkililerinizin son günlerde yaptığı tüm rütbe işlemlerini, toplam aktivitesini, kaç kişiyi yükseltip düşürdüğünü şık bir analitik panelde listeyerek yönetim performansını ölçün. Ayrıca performans ve güvenlik seviyesini de görebilirsiniz.
- Katı Hiyerarşi Koruması (/grup-maxrol): Yetkililerinizin komutla verebileceği maksimum rütbeyi sınırlandırarak alt yetkililerin üst rütbelere el uzatmasını tamamen engelleyin.
- Güvenli Kimlik Doğrulama (/doğrula): Topluluğunuzdaki üyelerin Roblox hesaplarını Discord hesaplarıyla eşleştirin, sunucu güvenliğini ikiye katlayın.
- Üst Düzey Güvenlik Protokolleri: Operasyon Kilidi (/operasyon-kilidi): Acil bir sızıntı durumunda tüm hiyerarşik akışı tek komutla dondurun.
